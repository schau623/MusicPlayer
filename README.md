# MusicPlayer
Custom music player made using Python. UI created using Tkinter, Python's standard GUI toolkit  
Functions:  
      - Play  
      - Pause  
      - Stop  
      - Add songs  
      - Skip song  
      - Prev song  
      - Remove songs from playlist   
      - Song position slider  
      - Volume slider  
      - Loop  
      - Playlist serialization  
      - Display song cover art
